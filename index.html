In 2015, I registered on github, mainly to take up continuing Microsoft's Font Validator, after a few months of private communication with Microsoft and Google. Microsoft eventually released the minimally fixed-up version, while I was already working on re-implementing the removed/withheld functionalities.

A few new repositories and enhancement forks of existing projects had spawned out of that effort. That includes SharpFont, mono, webkit-sharp, etc. The latest are mono-modification to address code-signing issues on Mac OS X, and ftx-darling for trying to run Apple's Font Tool Suite on Darling.

FontVal is a large effort and actively accepts donations. There are two links for donations via Paypal: link 1 asks for credit card details, link 2 is Paypal to Paypal, I believe. Please do not use the gmail account shown on the paypal links for correspondence - e-mails to it are not checked up to a few months.
